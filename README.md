# Task Management System

## Overview

A full-stack Task Management System built using Spring Boot, React, MySQL, JWT Authentication, and Swagger.

## Features

### Authentication

* User Registration
* User Login
* Password Hashing using BCrypt
* JWT Token Generation

### Task Management

* Create Task
* View Tasks
* Update Task
* Delete Task

### Frontend

* React.js
* Login Page
* Dashboard Page

### Backend

* Spring Boot
* Spring Security
* Spring Data JPA
* MySQL

### API Documentation

* Swagger UI

## Run Backend

```bash
mvn spring-boot:run
```

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

## Tech Stack

* Java
* Spring Boot
* MySQL
* React
* JWT
* Swagger
